# CodSoft Python Internship Tasks

This repository contains Python projects completed as part of the CodSoft internship program.

---

## 📌 Tasks Included

### 1. Simple Calculator 🧮
A basic calculator that performs:
- Addition
- Subtraction
- Multiplication
- Division

### 2. Password Generator 🔐
Generates strong random passwords using:
- Letters (uppercase & lowercase)
- Numbers
- Special characters

### 3. To-Do List App 📝
A command-line To-Do List that allows:
- Adding tasks
- Viewing tasks
- Removing tasks

---

## 🛠️ Technologies Used
- Python 3

---

## 📂 How to Run

Run each file separately:

```bash
python calculator.py
python password_generator.py
python todo_list.py
